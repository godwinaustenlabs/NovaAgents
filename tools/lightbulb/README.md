# Nova Tool - Lightbulb
A simple NAS-compliant tool that echoes back the provided input_text with a timestamp.

## Input Example
```json
{
  "id": "nova-lightbulb",
  "payload": {
    "input_text": "Hello world",
    "metadata": {
      "channel": "cli",
      "timestamp": "2025-08-14T12:00:00Z",
      "language": "en"
    }
  }
}
